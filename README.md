# World GDP by Countries Analysis

This project analyzes GDP of countries between the year 2020 and 2025 using Python, Pandas and SQL

## Features
- Data Cleaning
- Exploratory Data Analysis
- Visualization with Matplotlib and Seaborn

## Dataset
- Source: Global GDP by countries data analysis project (CSV format from Kaggle)
- Content: Countries with year from 2020 to 2025
- Size: 196 rows X 7 columns

## Tools and Libraries
- Python, SQL
- Pandas, Numpy -- Data cleaning and analysis
- Matplotlib, Seaborn -- Data Visualization
- Jupyter Notebook / Kaggle Notebook

## Key Insights
- Countries with the highest GDP 2020, 2022, 2024
- Countries with the lowest GDP 2020, 2022, 2024
- Top 10 Countries by GDP 2025
- Countries with Negative or stagnant GDP growth between 2020-2025
- Top 10 Countries with the fastest GDP growth between 2020-2025
